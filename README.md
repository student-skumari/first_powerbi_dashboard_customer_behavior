# first_powerbi_dashboard_customer_behavior
Data Analytics Project
Overview

This project demonstrates an end-to-end Data Analytics workflow. It covers data loading, data cleaning, exploratory data analysis (EDA), SQL querying, dashboard creation, and reporting. The goal is to transform raw data into meaningful insights using multiple analytical tools.

Dataset:

The dataset contains structured data in CSV/Excel format.

It includes fields such as customer details, transactions, and behavioral metrics.

Data was imported into Python and databases for analysis.
Tools & Technologies

Python – Data processing and analysis

Pandas / NumPy / Matplotlib / Seaborn – EDA and visualization

SQL (PostgreSQL / MySQL / SQL Server) – Data querying and manipulation

Power BI – Interactive dashboard creation

Jupyter Notebook – Development environment
How to Run
Python

Install required libraries:

pip install pandas numpy matplotlib seaborn sqlalchemy


Open Jupyter Notebook.

Run the .ipynb file step by step.

SQL

Import dataset into your database.

Execute SQL scripts provided in the /sql folder.

Power BI

Open the .pbix file.

Refresh data source if needed.

Conclusion

This project showcases practical data analytics skills including data preprocessing, SQL querying, visualization, and reporting. It reflects the ability to work with real-world datasets and deliver clear, business-oriented insights.
